Quero criar uma base de dados vetorizada (embeddings) de artigos científicos sobre etnobotânica - o uso de plantas por comunidades tradicionais em território brasileiro. Este sistema irá permitir o upload de artigos científicos em PDF, dos quais ele irá extrair os metadados (ano, autores, titulo, resumo etc), que serão armazenados em um banco de dados SQL, e listados em uma página de consulta. Além disto, o sistema irá vetorizar o conteúdo destes artigos para uma base de dados vetorial, usando um modelo de "embeddings" voltado para artigos científicos, preferencialmente gratuito e open source (pesquisar em https://huggingface.co/). Além disto, o sistema oferecerá uma interface (chat) para sistema de busca inteligente onde quero fazer consultas como "plantas usadas para inflamação em comunidades amazônicas" ou "espécies com propriedades anti-parasitárias validadas cientificamente" e encontrar artigos semanticamente relevantes, mesmo que não contenham exatamente essas palavras. O sistema poderia também ofertar uma interface para um um sistema de recomendação que, dado um artigo, sugere outros relacionados de forma não-óbvia — conectando pesquisas de diferentes regiões que estudam plantas com propriedades similares, por exemplo; e também para análise de tendências: identificar quais plantas estão sendo mais estudadas, lacunas em pesquisas, ou padrões entre comunidades diferentes. Este sistema deve considerar os princípios C.A.R.E para seus dados, e rodar em um docker. A interface de "chat" deve permitir a associação de chaves para API de agentes como Gemini, Claude e ChatGPT, por usuário. Busque as melhores opções de modelos de "embeddings" e bancos de dados vetoriais e SQL gratuitos, ou com melhor custo-benefício. Crie um readme.md destacando os benefícios acadêmicos e para as comunidades tradicionais de um banco de dados vetorizado sobre etnobotânica, baseado em artigos científicos. O número de artigos científicos esperado pode chegar as dezenas de milhares. O sistema deve prever conexão com APIs de revistas científicas para monitorar o surgimento de novas publicações científicas sobre o tema e adicionar os metadados na base de dados SQL e vetorizar seu conteúdo, automáticamente.
